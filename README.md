# campus-network-public

#### 介绍
广工校园网登陆工具-开源

#### 软件架构
使用golang编写


#### 安装教程

1.  下载最新发行exe，或下载源代码自己编译
2.  在同级目录下新建“config.txt”文件
3.  在文件中第一行填写校园网登录username，第二行填写password

#### 使用说明

1.  确保自己连上gdut后(2.0版本后只需打开wifi即可) 
2.  双击exe文件
